# HouHou Fighter v1.1.0 구조 정리

## 목적
캐릭터가 늘어나도 `game.bundle.js`에 직접 조건문을 계속 추가하지 않도록, 전투 계산과 스킬 처리를 Manager 구조로 분리했다.

## 핵심 구조
- `character_manifest.json`: 로드할 캐릭터 목록
- `assets/characters/<key>/character.json`: 캐릭터 스탯/크리/스킬/이펙트 설정
- `DamageCalculator`: 공격력, 방어력, 크리티컬 배수 계산
- `SkillManager`: 흡혈, 2타, 마비 같은 스킬 후처리
- `EffectRenderer`: 공격 이펙트, 좌우 반전, 중심/전방 오프셋 처리

## 신규 스킬 추가 방법
1. `SkillManager`에 처리 로직 추가
2. `character.json`의 `skills` 배열에 스킬 type과 옵션 추가

예시:
```json
{
  "type": "poison",
  "chance": 30,
  "damage": 5,
  "duration": 3,
  "tick": 1
}
```

## 화면 표시
시작 하단 우측에는 `Art by 홍은성`을 유지한다.
