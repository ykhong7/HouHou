# v1.2.0 CharacterManager

이번 버전부터 캐릭터 로딩/정규화/가챠풀 생성을 `js/managers/CharacterManager.js`가 담당합니다.

## 처리 범위
- `data/character_manifest.json` 읽기
- 각 캐릭터 `character.json` 읽기
- 기존 평면 구조와 신규 Data-Driven 구조 모두 정규화
- `stats`, `critical`, `gacha`, `combat`, `assets`, `effect`, `projectile`, `skills` 해석
- `data/gacha.json`이 없거나 누락되면 캐릭터의 `gachaRate`로 자동 생성
